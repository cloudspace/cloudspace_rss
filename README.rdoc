== Project Setup Instructions

<b>Create and launch the vagrant box</b>
1. git clone git@github.com:cloudspace/cloudspace_rss
2. cd ./cloudspace_rss
3. vagrant up

<b>After the box is up, configure it</b>
1. vagrant ssh
2. sudo apt-get install libmagickwand-dev libsasl2-dev #needed for rmagick and memcached, respectively
3. sudo gem install bundler
4. cd /srv/cloudspace_rss
5. bundle
6. rake db:setup
7. rails s #start the rails server
